# 🐉 Alexa AI

> Asisten kecerdasan buatan lokal yang cantik dan powerful — didukung oleh **Ollama** & **OpenClaw**.

![Alexa AI Logo](public/logo.png)

## ✨ Fitur

- 🤖 **AI Chat Real-time** — Streaming response dari model OpenClaw via Ollama
- 💬 **Multi-Conversation** — Buat, kelola, dan hapus percakapan
- 💾 **Database Lokal** — Semua chat tersimpan di localStorage browser
- 🎨 **UI Cantik** — Dark theme dengan aksen pink naga, animasi heartbeat
- 📱 **Responsive** — Berfungsi di desktop dan mobile
- ⚡ **100% Lokal** — Tidak ada data yang dikirim ke cloud
- 🐳 **Docker Ready** — Deploy dengan satu perintah

## 🛠️ Tech Stack

| Teknologi | Kegunaan |
|-----------|----------|
| React + TypeScript | Frontend |
| Vite | Build tool |
| Tailwind CSS | Styling |
| Framer Motion | Animasi |
| Ollama | LLM Runtime |
| OpenClaw | Model AI |
| Docker | Deployment |

## 🚀 Quick Start

```bash
chmod +x deploy.sh
./deploy.sh
```

Baca [deploy.md](deploy.md) untuk panduan lengkap.
